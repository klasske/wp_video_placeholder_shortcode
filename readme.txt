Ligthweight shortcode plugin that avoids preloading videos from sites such as Mediasite by loading an image placeholder instead

== Description ==

Add your shortcode in the following format:

	[video_placeholder src="http://example.com/placeholder_image.jpg" 
	videosrc="http://example.com/the_video_url" caption="caption" align="left"]
